Please Note

This app is not affliated with Canvas or Instructure. This is a completely independent, third-party application developed by a solo developer. This app uses the Canvas LMS API solely to retrieve information such as your courses, assignments, name, and profile icon. No personal data is ever collected or sent to a server. Canvas Credentials are securely saved on-device and all user settings are saved locally. Please verify all assignment details, as Canvas HW Reminder is not responsible for any missed deadlines.

If you encounter any issues, Please contact Kartik at "yg.kartik@gmail.com".
